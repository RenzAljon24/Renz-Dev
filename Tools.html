Social Apps and Tools I've Used

I've found several social apps and tools to be incredibly useful in enhancing my learning experience. Here are some of the key ones:

  • LinkedIn: For tech-related posts and engagements, it's like the Facebook version of tech people.
  • GitHub: This is where I deploy my projects and manage version control.
  • Quizlet: Study and Flashcard Creation
  • Stack Overflow: This is by far my favourite platform when I'm coding. It's where I go to find answers and solutions. It's like my quick guide to getting things done while coding, and it's always there when I need it.
  • Adobe Photoshop: I rarely use this, but it is useful when you are designing your website or just want to edit and design some stuff.

Flowchart and Diagram Tools

  • Lucidchart: Web-based tool for collaborative flowcharts, mind maps, and UML diagrams.
  • Draw.io: Free, open-source online tool for creating flowcharts, network diagrams, and more.
  • SmartDraw: Diagramming tool with integration options for various platforms.

Online learning Platforms

  • W3schools: This site is very helpful when you are a beginner. It provides step-by-step explanations and free code samples to experiment with and make changes to.
  • Udemy: Udemy is another great site. They offer free courses with certificates that you can take advantage of.

Using these social app tools and online learning platforms has really helped me in my coding and technology journey.
